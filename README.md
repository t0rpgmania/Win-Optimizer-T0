# 🇧🇷 Otimizador Mestre - Performance e Latência

Este projeto é um script de otimização para Windows focado em reduzir a latência e maximizar o desempenho em jogos, sem remover serviços essenciais do sistema.

## ⚠️ Aviso Legal (Disclaimer)
Este software é fornecido "como está", sem qualquer garantia. O uso deste script é por conta e risco do usuário. O autor não se responsabiliza por eventuais perdas de dados ou instabilidades no sistema.

## ⚙️ Funcionalidades
- **[1] Otimização Eficiente:** Ajustes de registro focados em menor consumo de energia e latência otimizada.
- **[2] Otimização Turbo:** Foco total em desempenho máximo, ignorando estados de economia de energia.
- **[3] Verificar e Reparar Windows:** Executa a varredura nativa `sfc /scannow`.
- **[4] Centro de Verificação:** Ferramenta integrada para abrir registros e serviços do Windows, permitindo total transparência sobre as alterações feitas.
- **[5] Reversão Segura (Reset):** Restaura todas as configurações de performance aos padrões originais do Windows.
- **[6] Módulo Avançado de Compressão (NOVO v5.0):** Bloco isolado focado em liberar espaço e otimizar memória:
  - **RAM:** Ativa compressão dinâmica para reduzir o uso do arquivo de paginação (Pagefile).
  - **Jogos (XPRESS8K):** Compacta pastas no HD/SSD reduzindo o tamanho em disco sem causar perda de FPS, usando compressão nativa do Windows.

## 🚀 Como usar
1. Baixe o arquivo `.bat` mais recente na aba **Releases**.
2. Clique com o botão direito e selecione **"Executar como Administrador"**.
3. Siga as instruções no menu.

# 🇺🇸 Master Optimizer - Performance & Latency (English Guide)

*Note: The `.bat` script interface is kept in Portuguese (PT-BR) to prevent syntax collisions and bugs within the Windows CMD interpreter. Below is the full English translation of its features so you can navigate it easily.*

This project is a Windows optimization script focused on reducing latency and maximizing gaming performance without removing essential system services.

## ⚠️ Disclaimer
This software is provided "as is", without warranty of any kind. The use of this script is at your own risk. The author is not responsible for any data loss or system instability.

## ⚙️ Features (Menu Translations)
- **[1] Efficient Optimization (Consumo Mínimo):** Registry tweaks focused on lower power consumption and optimized latency.
- **[2] Turbo Optimization (Consumo Máximo):** Full focus on maximum performance, ignoring power-saving states.
- **[3] Verify and Repair Windows:** Runs `sfc /scannow` to fix corrupted system files.
- **[4] Verification Center (Ver Mudanças):** Built-in tool to open Windows Registry and Services, allowing full transparency regarding what the script changes. Includes a tutorial on default vs. modified values.
- **[5] SAFE REVERT (Reset Padrão):** Restores all settings modified by options 1 and 2 back to Windows defaults.
- **[6] ADVANCED COMPRESSION MODULE (RAM & Folders - NEW v5.0):** An isolated block for storage and memory management:
  - **RAM Compression:** Enables dynamic Windows memory compression to reduce Pagefile usage.
  - **Game Folder Compression:** Compresses game folders directly on your HDD/SSD using the native **XPRESS8K** algorithm (Xbox standard). Saves space with ZERO FPS loss or stuttering.
  - **Decompression:** Reverts folders back to their raw size.

## 🚀 How to use
1. Download the latest `.bat` file from the **Releases** tab on the right side of the screen.
2. Right-click the file and select **"Run as Administrator"**.
3. Follow the menu instructions (use the translation above as a guide).

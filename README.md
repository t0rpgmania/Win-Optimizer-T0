# Otimizador Mestre - Performance e Latência

Este projeto é um script de otimização para Windows focado em reduzir a latência e maximizar o desempenho em jogos, sem remover serviços essenciais do sistema.

## Funcionalidades
- **Otimização Eficiente**: Ajustes de registro focados em menor consumo de energia e latência otimizada.
- **Otimização Turbo**: Foco total em desempenho máximo, ignorando estados de economia de energia.
- **Centro de Verificação**: Ferramenta integrada para abrir registros e serviços do Windows, permitindo total transparência sobre as alterações feitas.
- **Reversão Segura**: Opção para restaurar todas as configurações aos padrões originais do Windows.

## Como usar
1. Baixe o arquivo `.bat`.
2. Clique com o botão direito e selecione **"Executar como Administrador"**.
3. Siga as instruções no menu.
